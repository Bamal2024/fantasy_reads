# Fantasy Reads

Full Stack web application built with Django.

## Overview
Fantasy Reads is a personal project focused on content management,
user experience, and clean separation between backend logic and frontend presentation.

The project includes:
- Book catalog
- Categories and merchandising
- Admin customization
- Media handling
- User-facing views

## Tech Stack
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite (development)

## Features
- Dynamic content rendering
- Django Admin customization
- Structured templates and static files
- Media handling for covers and products

## Status
Project developed as part of formal programming studies.
Further improvements and deployment planned.
